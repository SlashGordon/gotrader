# gotrader
Alpaca trader

## config

Add a file named `.gotrader.yaml` to your workspace with the following content:

```yaml
APCA_API_KEY_ID: "alpacaid"
APCA_API_SECRET_KEY: "alpacakey"
APCA_API: "https://paper-api.alpaca.markets"
`````
